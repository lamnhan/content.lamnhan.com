### Cài đặt

Cài đặt CLI chung.

```sh
npm i -g @lamnhan/seminjecto
```

Hoặc, cài riêng theo dự án:

```sh
npm i -D @lamnhan/seminjecto
```

### Sử dụng

Tạo thư viện Node:

```sh
semidi new lib myLib
```

Tạo ứng dụng CLI:

```sh
semidi new cli myCli
```
