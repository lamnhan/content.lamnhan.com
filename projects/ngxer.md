### Installation

Install the CLI globally:

```sh
npm i -g @lamnhan/ngxer
```

Or, locally:

```sh
npm i -D @lamnhan/ngxer
```

### Usage

Add and edit configuration in the file `.ngxerrc.json`.

```sh
ngxer init
```

Then, generate static files:

```sh
ngxer generate
```

View report after rendering:

```sh
ngxer report
```
