### Installation

Install as glocal CLI app.

```sh
npm i -g @lamnhan/ayedocs
```

Or locally:

```sh
npm i -D @lamnhan/ayedocs
```

### Usage

Create file and edit the configuration in the file: `.ayedocsrc.js`

Generate document:

```sh
ayedocs generate
```
