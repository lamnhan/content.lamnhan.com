### Cài đặt

Cài đặt CLI chung.

```sh
npm i -g @lamnhan/ayedocs
```

Hoặc, cài riêng cho từng dự án:

```sh
npm i -D @lamnhan/ayedocs
```

### Sử dụng

Tạo tệp và chỉnh sửa những thông số trong tệp tùy theo dự án: `.ayedocsrc.js`

Tạo tài liệu tự động:

```sh
ayedocs generate
```
