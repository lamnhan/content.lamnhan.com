### Cài đặt

Cài đặt CLI chung:

```sh
npm i -g @lamnhan/ngxer
```

Hoặc, riêng cho từng dự án:

```sh
npm i -D @lamnhan/ngxer
```

### Sử dụng

Thêm tập thiết lập và chỉnh sửa thiết lập trong tệp `.ngxerrc.json`.

```sh
ngxer init
```

Tiếp theo, xuất các tệp HTML:

```sh
ngxer generate
```

Xem báo cáo các thông tin đã xuất:

```sh
ngxer report
```
