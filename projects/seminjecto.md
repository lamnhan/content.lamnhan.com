### Installation

Install as glocal CLI app.

```sh
npm i -g @lamnhan/seminjecto
```

Or locally:

```sh
npm i -D @lamnhan/seminjecto
```

### Usage

Create Node library:

```sh
semidi new lib myLib
```

Create Node CLI:

```sh
semidi new cli myCli
```
